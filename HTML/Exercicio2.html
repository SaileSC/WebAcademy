A Linguagem HTML

HTML (abreviação para expressão inglesa HeperText Markup Language, que significa Linguagem de Marcação de Hipertext) é uma linguagem de marcação utiluzada na contrução de páginas na Wrb. Documentos HTML podem ser interpretados por navegadores

Alguns dos recursos da linguagem HTML vistos até este momento:

Recursos Tags associadas Descrição
Links <a> Criar links entre paginas
Listas <ul> e <li> Criar lostas de itens ordenados
<ol> e <li> Criar listas de itens nao ordenados
Imagens <img> Inserir imagens
